# BYREE RATHAN KESHAV - Portfolio

A modern, premium, fully responsive personal portfolio website showcasing my skills, projects, and professional journey as a Computer Science student.

## Description

This portfolio website is designed to present my professional profile in an elegant and interactive way. Built with modern web technologies, it features smooth animations, responsive design, and a premium dark teal and neon cyan color scheme that creates a memorable user experience.

## Features

- **Responsive Design**: Fully responsive across mobile, tablet, and desktop devices
- **Modern Hero Section**: Unique layered text animations with floating particle effects and scroll indicator
- **Smooth Scrolling Navigation**: Seamless navigation between sections with active state indicators
- **Skills Animation**: Progress bars that animate when scrolled into view using Intersection Observer
- **Contact Form**: Functional contact form that opens email client with pre-filled information
- **Social Media Integration**: Direct links to GitHub, LinkedIn, X (Twitter), and Gmail with genuine X logo
- **Modern UI/UX**: Premium dark teal and neon cyan design with gradients, glow effects, and smooth transitions
- **Interactive Elements**: Hover effects, scale animations, and micro-interactions throughout
- **Professional Typography**: Clean and modern fonts using Google Fonts (Poppins)

## Technologies Used

- **HTML5**: Semantic markup for structure and accessibility
- **CSS3**: Modern styling with gradients, animations, and transitions
- **Bootstrap 5**: Responsive grid system and components
- **JavaScript**: Vanilla JS for interactivity and animations
- **Font Awesome**: Icon library for social media icons
- **Google Fonts**: Poppins font family for modern typography

## Setup Instructions

1. **Clone or Download**: Get the project files to your local machine
2. **Add Profile Image**: Place your profile image as `rathan-keshav.jpg` in the root directory
3. **Open in Browser**: Open `index.html` in your preferred web browser

### Quick Start

```bash
# Navigate to the project directory
cd portfolio

# Open index.html in your default browser
# Double-click the file or use:
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

## Customization

### Colors
The color scheme can be customized by modifying the CSS variables in `style.css`:

```css
:root {
    --primary-dark: #0a2e2e;
    --primary-teal: #008b8b;
    --neon-cyan: #00ffff;
    /* ... other variables */
}
```

### Content
- Update personal information in `index.html`
- Replace `rathan-keshav.jpg` with your profile image
- Modify skills and percentages in the skills section
- Update social media links in the contact section
